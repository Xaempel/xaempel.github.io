Fundamentals of C++

Beginning

To begin writing C++ code, all you need is a C++ compiler. Two popular choices are g++ (part of GCC) and Clang. I won't provide a tutorial on how to install a compiler here — it's easy to find one online, or you can use an online C++ compiler to get started quickly. If you already have a compiler, let's begin!

First Step into C++ - Lasciate ogni speranza, voi ch'entrate

The first program that almost everyone writes when learning a programming language is the classic Hello World app. This program is very simple because it contains only one instruction: it displays text on the terminal screen. In C++, such a program looks like this: 


            #include <iostream>

            int main() {
                std::cout << "Hello, World\n";
                return 0;
            }

At first glance, this code might seem intimidating, but it's actually very straight forward. Let's break it down line by line:

  • #include <iostream> - In short, #include allows us to use predefined types, functions, or algorithms from the standard library. In this case, we are adding basic input/output functionality to our program.
  • int main() - This is the entry point of the program. The curly braces {} enclose the code that will be executed when the program runs.
  • std::cout << "Hello World" ; - This instruction displays text on the terminal. The std::cout << part sends the text inside the quotes ("") to the output.
  • \n - This symbol tells the program to insert a new line into the output.

Next Steps in C++ - Variables and Basic Arithmetic Operators

Now that you know how to print output to the terminal, the next step is to learn about variables and arithmetic operations. A variable is like a box where you can store data. In C++, there are many different types of variables, but to get started you only need three or four basic ones: int, float, char, and std::string.

  • int - stores integer values like 9, 12, 32, 10512.
  • float - stores floating-point values like 1.23, 125.125, 512.3.
  • char - stores a single character like 'a', 'b', 'y'.
  • std::string - stores a sequence of characters (text). To use strings, you must include the <string> header.
Here's an example: 
            
                #include <iostream>
                #include <string>

                int main() {
                    int a {5};
                    float b {3.12};
                    char ch {'A'};
                    std::string sentence {"This is a sentence"};
                }
Explanation:
  • #include <iostream> - You already know this from the previous example.
  • #include <string> - Adds support for using std::string.
  • int a, float b, char ch – Declare variables of types int, float, and char.
  • std::string sentence - Declares a variable of type std::string. The std:: prefix means the type comes from the C++ standard library (just like std::cout). This example shows how to initialize variables in C++ using curly braces {}. This method is preferred in modern C++ because it is safer and more predictable than using =. You can also write: 
                        
                        int number = 5;
                        float pi = 3.14;
    but {} is recommended.

Printing Variables

You can print variables to the terminal like this: 

                
                #include <iostream>
                #include <string>

                int main() {
                    int a {5};
                    // Prints 5 
                    std::cout << a << "\n";

                    a = 2;
                    // Prints 2
                    std::cout << a << "\n";

                    float b {3.15};
                    // Prints 3.15
                    std::cout << b << "\n";

                    std::string str {"This is a sentence"};
                    // Prints "This is a sentence"
                    std::cout << str << "\n";
                }
Key points:
  • // - Starts a comment. Comments are for programmers and are ignored by the compiler.
  • std::cout << a << "\n"; - Prints the value of a and moves to the next line.
  • a = 2; - Assigns a new value to a variable. You can do the same with float or std::string.

Arithmetic in C++

Now let's look at some basic arithmetic operations:

            
                #include <iostream>

                int main() {
                    int a {6};

                    // Addition
                    a = a + a; // 6 + 6 = 12
                    std::cout << a << "\n";

                    // Subtraction
                    a = a - a; // 6 - 6 = 0

                    // Multiplication
                    a = 6;
                    a = a * a; // 6 * 6 = 36

                    // Division (integer division)
                    a = 6;
                    a = a / 4; // Result: 1 (fraction is discarded)

                    // Modulo (remainder)
                    a = 6;
                    a = a % 4; // Result: 2
                }

These are the fundamental arithmetic operations in C++: addition (+), subtraction (-), multiplication (*), division (/), and modulo (%). They will be essential for writing your first simple programs.

How to Get Input from the User in C++

This is the last topic I'll cover in this article. Once you understand it, you'll be able to write simple programs — still without logic statements, but we'll explore those in the next article. So, let's get to the point. 

            
                #include <iostream>

                int main() {
                    // Create a variable to store the user's input
                    int a {0};

                    // Get input from the user
                    std::cin >> a;

                    // Display the value entered by the user
                    std::cout << a;
                }

Explanation:

  • In this example, we use two functions you already know or have seen before:
    • std::cout - prints output to the terminal.
    • std::cin – gets input from the user via the terminal.
  • The >> operator with std::cin stores the entered value into the variable on the right-hand side.
  • In this example, we store the input in an int variable called a and then immediately print it back.

Summary

You know now:

  • How to use basic input and output in C++.
  • What variables are and how to declare them.
  • How to perform basic arithmetic operations.

With these fundamentals, you can already write small C++ programs. The next step is to learn logic statements like if/else and loops — the building blocks of more complex applications. See you in the next part!
